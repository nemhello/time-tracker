## What Each Button Does:

### Edit Button (On Each Entry)
**Click it to:**
- Adjust start time
- Adjust end time
- Round times for cleaner records
- Fix mistakes

**Example:**
```
Original:  08:47 - 12:23  (3h 36m)
After Edit: 08:45 - 12:30  (3h 45m)
```

### Export Backup Button (Bottom of Home Screen)
**Click it to:**
- Download all your entries
- Save as JSON file
- Backup before updates
- Keep records safe

**Creates file:**
`time-tracker-backup-2024-12-11.json`

### Import Backup Button (Bottom of Home Screen)
**Click it to:**
- Restore from backup file
- Import after reinstalling
- Recover lost data
- Merge old entries

**Prompts you to:**
Select backup file â†’ Confirm import â†’ Done!
